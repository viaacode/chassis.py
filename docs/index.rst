Chassis.py
==========
Chassis.py is a framework/library that helps developing services
by providing a solution for a some cross-cutting concerns.

You can install the chassis using pip.

.. code-block:: console

    $ pip install viaa-chassis

.. note::
    viaa-chassis is only available on VIAA's internal PyPI.
